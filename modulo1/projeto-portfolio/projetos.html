2. Criação do jogo BlackJack

Este projeto foi um exercício proposto pela Labenu, nele, criei um jogo semelhande ao BlackJack. O jogo tem como participantes o usuário e o computador, ao iniciar um partida é sorteado 2 cartas de um baralho para cada jogador, vence quem possuir a soma dos valores das cartas maior.

As ferramentas que usei para a criação do jogo foram:

- HTML

- JavaScript
